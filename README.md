# Front end: ReactJS app

## Dependencies used

- "prop-types": "^15.7.2",
- "react": "^16.12.0",
- "react-dom": "^16.12.0"
- "gh-pages": "^2.1.1",

## Getting started

1. Clone the project into your machine and install all dependencies inside the source folder using:

```console
yarn install
```

2. Now, run the server and see the result in http://localhost:8080/:

```console
yarn dev
```
